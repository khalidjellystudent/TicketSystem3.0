Common fixes
- Red editor page (Razor parse error): look for nested double quotes in attributes (use single quotes for data URLs).
- Password eye: Login.cshtml uses #Password and #togglePassword script — increase input padding (pr-16) so icons don't overlap.
- Sidebar overlap (mobile): sidebar is off-canvas; use hamburger to open/close.
